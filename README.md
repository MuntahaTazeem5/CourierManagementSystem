Courier Management SystemA lightweight, desktop-based Courier Management System built using Java Swing (GUI). This application allows users to log courier details, track shipping statuses via manual entry, and easily manage shipping records through a user-friendly interface.  🚀 FeaturesCourier Registration: Easily input details like Sender Name, Receiver Name, Destination Address, Tracking ID, and Shipping Charges.  Real-time Status Tracking: Update and display custom delivery statuses (e.g., Delivered, On the Way) instantly.  Form Reset: A quick-clear feature to wipe the entry fields and start a new transaction instantly.  Simple Desktop GUI: Built entirely using native Java standard libraries (javax.swing and java.awt).  🛠️ Tech StackLanguage: Java  GUI Framework: Java Swing & AWT  IDE Framework: Designed using NetBeans / Standard Java Environment  📋 Interface OverviewThe application includes a single, clean window interface with the following parameters:  ComponentDescriptionInput FieldsSender Name, Receiver Name, Address, Tracking ID, and Charges.  Add Courier ButtonSaves the courier data in memory and outputs a success confirmation.  Tracking Status ButtonPrompts an input dialogue to update and view the full delivery summary.
